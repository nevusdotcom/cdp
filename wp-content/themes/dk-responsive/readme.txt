DK Responsive Features
-------------------------------------------------------

The DK Responsive is simple wordpress theme. Theme have Theme Options with Logo upload, favicon upload, Social media,Home Page Callout Settings,Layout Settings, Slider settings etc. Theme Features 3 template layouts, 
6 widget areas. The DK Responsive theme is W3C Markup Validated, RTL-Language Support and Mobile Friendly.
